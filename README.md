Download XLAND operating system files

In this source, which is the legal source of XLAND, you can download the XLAND operating system files and easily create your own operating system. You can create XLAND distributions and easily create your own operating system. This feature that the XLAND team has given you can do without writing a bootloader or kernel. You can use the ready-made kernel and bootloader made by the XLAND team. But pay attention to not posting any changes you make to the bootloader on the Internet because this ready-made bootloader and ready-made kernel were made by the XLAND team and this operating system is nothing. You can create your own personal desktop and create your own ISO. Note that all the bootloader and kernel codes made by the XLAND team are pure ASM language.
